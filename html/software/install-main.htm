System Requirements

The following instructions assume you are familiar with the Linux Operating System.

Operating System

You will need one of the following distributions of Linux:

  • Red Hat Enterprise Linux 3
  • Red Hat Fedora Core 3

Processor

In additional to the minimum processor required by the operating system you are running, this CD requires a 500 MHz Pentium III processor. Sun® Java™ Studio Creator 2 Update 1 requires a 1 GHz Intel Pentium 4 processor (or equivalent).

Graphics Display Card / Sound Card

16-bit color mode with KDE or the Gnome desktop and in conjunction with displays set to local hosts. A standard graphics display card, with a minimum screen resolution of 1024x768 is adequate.

RAM

In additional to the minimum RAM required by the operating system you are running, this CD requires a minimum of 512 MB of RAM. Sun Java™ Studio Creator 2 Update 1 requires 1 GB of RAM.

Disk Space

Your computer must have a minimal of 1.5 GBs of hard disk space available. You may be able to make more space available by removing temporary files on your computer, including temporary files created when you decompress some of the software on this CD prior to installing it.

Additional Software

Web browser, Adobe® Acrobat® Reader® (7.0.1) and a Zip decompression utility. All three are typically included with current Linux distributions.

If you do not have Adobe Acrobat Reader; installed on you computer, check on the latest version of Acrobat Reader at...

http://www.adobe.com/products/acrobat/readstep.html

Click on each software titles in the left window to read the installation instructions. Even though all the software is distributed on this CD, you may need to connect to the Internet in order to access a number of important links in the installation instructions.

Be sure to pay careful attention to the software requirements relative to prerequisite programs that must be installed prior to proceeding with installing the software packages on this CD.

NOTE: If at any point while installing the software supplied on this CD, you are presented with an error message that states you do not have the correct privileges or permissions to install the software, you will need to log off and then log on as the root user (or as a user with the correct permissions, a.k.a. superuser). If you are installing this software at a company, educational institution, or other organizational site, you may need to ask your system administrator for assistance or for an appropriate password.

Software Environment Requirements

In order to install and use several of the software programs included on this CD, you must have certain prerequisite software installed. In most cases this software is provided either as a separate software package or as a component of one of the packages. Some of the requirements, however, are not included on the CD.

The following table outlines the dependencies of the various software programs on one another and on other programs. You should read this now to get an overview and re-read it before installing any of the software on this CD. Come back to this table again to troubleshoot any installation issues that might arise. Be sure read the annotations which provide more details about many of these requirements. In each case, it is highly recommended that you view both programs' homepages to determine what has changed between the pressing of the CD and your purchase of the textbook.
