System Requirements

This CD includes the installer for MySQL and requires the following...

  • Most recent (6.0) version of Sun's Java™ 2 SDK installed, prior to installing this software.
  • At the minimum you will need a version of the Linux OS that is built with on kernel 2.0 to install MySQL.
  • You need one of the following tools to uncompress the MySQL tar file binary distribution. GNU gunzip or a reasonable tar utility; GNU tar is known to work. Either is usually pre-installed on most versions of the Linux operating system.
  • Unix socket file connections preferred over TCP/IP connections.

Installing the MySQL Binary on Linux

The instructions that follow are for installing the binary distribution of MySQL. For alternative installation packages of the MySQL Community Server please visit the MySQL 5.0 Downloads site. You can refer Chapter 2 of the MySQL 5.0 Reference Manual covers the installation process of the alternative packages.

You must have the most recent (6.0) version of Sun's Java™ 2 SDK installed, prior to installing this software.

Navigate to the directory (software\NetBeans\) on this CD that contains the MySQL installer (mysql-standard-5.0.27-linux-i686.tar.gz) and download the file into a writable directory on your system.

The basic commands that you must execute to install and use a MySQL binary distribution are:

  1. To extract the archive mysql-standard-5.0.27-linux-i686.tar.gz into the same directory: 
    tar xzf mysql-standard-5.0.27-linux-i686.tar.gz

    If this fails, the version of tar may not support gzip compression. In this case, you can use the traditional two-stage command: 

    gzip -dc mysql-standard-5.0.27-linux-i686.tar.gz | tar xf -
  2. Once the binary is extracted, please do the following to install the MySQL binary... 
    shell> ln -s FULL-PATH-TO-MYSQL-VERSION-OS mysql
shell> cd mysql
shell> scripts/mysql_install_db --user=mysql
shell> chown -R root  .
shell> chown -R mysql data
shell> chgrp -R mysql .
shell> bin/mysqld_safe --user=mysql &

NOTE: This procedure does not set up any passwords for MySQL accounts. Refer to the Post-Installation Procedures section of the MySQL 5.0 Reference Manual to customize your installation of MySQL to meet you needs.

For a more detailed installation alternative refer to Chapter 2 of the MySQL 5.0 Reference Manual
